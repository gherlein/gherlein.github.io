VLC Error

03 Oct 2021, 00:00
technical / linux / vlc

A bug in VLC? Grrrr. Reminds me of the old days.

I’m playing with my video cameras this morning and moved some displays to a new computer with a fresh install of Ubuntu. And VLC is broken. It just won’t play the stream on the new machine. Works like a champ on the older one. Someone else had this problem, it seems, so I added to the github issue. I have a sneaky suspicion this is a regression.
