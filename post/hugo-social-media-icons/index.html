Fixing Hugo Social Media Icons in Blackburn

26 Sep 2020, 00:00
technical / linux / hugo / blogging / blackburn

The Blackburn theme for Hugo has a bug in how it shows social media icons. Here’s how to fix it.

My friend David Wang also uses Blackburn and he shared with me his solution.

TL;DR

Until the PR merges into Blackburn:

cd themes/blackburn/layouts/partials
rm share.html
wget https://raw.githubusercontent.com/yoshiharuyamashita/blackburn/186f596907bd6ab0c746631bf8b66978b8072eef/layouts/partials/share.html

Rebuild. Worked for me.
