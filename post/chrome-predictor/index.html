Changing Chrome Address Bar Suggestions

  • 03 Aug 2024, 00:00
  • general

Leaving this here mostly for me, but short instructions on how to change Chrome’s “typeahead” predictors if it remembers the “wrong” one.

Chrome “remembers” where you visit and uses that as a “typeahead” shortcut. I use this all the time. Sometimes if you do a lot with one location it remembers THAT one but it’s not the general one you really want to be remembered. And I’m always forgetting how to clear it. So here’s a note to my future self, and anyone else who it can help.

TL;DR: hightlight the offending URL in the address bar, and then press:

  • Mac: Shift+FN+Delete
  • Linux and Windows: Shift + Delete
  • Chromebook: Alt + Shift + Delete
