Make RDP work from OSX

21 Apr 2020, 06:10
technical / linux / rdp / osx

Quick reminder how to enable RDP sessions from OSX to an Ubuntu box.

Stupid little thing, but when you want to use your MacBook and remote desktop to an Ubuntu notebook - I know, that’s kinda rare - but alas. Using the built in OSX screen sharing app will work, but Ubuntu needs a tweak from the command line to disable encryption:

gsettings set org.gnome.Vino require-encryption false

Credits to a reddit post

These things are too hard for normal people, and I suspect it’s all around making it “easy” to just stay all within one ecosystem.
